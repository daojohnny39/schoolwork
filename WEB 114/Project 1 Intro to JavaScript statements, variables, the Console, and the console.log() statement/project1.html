What is the purpose of JavaScript? Answer in general and also provide three specific examples of how JavaScript is commonly used.

Most websites use JavaScript for web page interactivity. This could mean a variety of things such as instant user feedback, pop-up materials, and interative forms. These functionalities gives the illusion that the web page is "more alive".

Another way that websites use JavaScript is to dynamically update web pages. Imagine a site like Amazon where the content is constantly being updated as the user scrolls. This function relies on JavaScript.

JavaScript can also be used for web application development. JavaScript is the foundation of many different web frameworks like React and Angular. These different frameworks allows the developer to create a website that closely resembles an actual desktop application.

What's the difference between how you reference a variable versus a string in a JavaScript statement?

Generally, when you reference a string, you should use either "" or '' around the text. As for integers, you can leave them as is.

For example:

let x = "Hello";

What's the difference between a single value and an expression in a JavaScript statement?

When assigning a single value to a variable, there are no calculations done. When assigning an expression to a variable, there are calculations being done whether it be mathematical or just combining strings.
